# Lost and Found Campus System

A React-based web application that helps students report lost items and found items within a campus.

## Features
- Report lost items
- Report found items
- Claim found items
- Admin management system

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS

## How to Run the Project

1. Clone the repository
2. Install dependencies
3. Start the application
     application wll run on:
       http://localhost:3000
